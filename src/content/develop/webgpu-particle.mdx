GPGPU curl noise particle simulation built with WebGPU + Three.js TSL.

65,536 particles flowing through a divergence-free curl noise field. Mouse interaction creates real-time repulsion effects.

## Tech Stack

- WebGPU Compute Shaders (GPGPU)
- Three.js TSL (Three Shading Language)
- Vite + TypeScript
